# 《决胜21点》 — 游戏概念文档

## 概述

《决胜21点》是一款基于传统21点规则的策略卡牌对战游戏。在保留21点核心玩法（要牌、停牌、爆牌）的基础上，引入花色战斗效果、牌型倍率、卡牌印记与卡质系统，以及商店构筑机制，打造兼具运气与策略深度的 Roguelike 卡牌体验。致敬《小丑牌》(Balatro)。

## 类型与标签

- **类型**: 策略卡牌 / Roguelike 构筑
- **视角**: 2D 俯视牌桌
- **参考作品**: 小丑牌 (Balatro), 21点 (Blackjack)
- **目标平台**: PC (Steam / Epic)
- **玩家人数**: 单人对战 AI
- **单局时长**: 约 20-40 分钟

## 核心循环

```
对战回合 → 结算 → 商店构筑 → 下一回合 → ... → 击败最终对手
```

1. 玩家与 AI 对手各持一手牌，逐张结算，花色决定效果
2. 回合结束后进入商店，用筹码购买卡牌、印记、卡质
3. 逐步构筑更强的牌组，击败一系列对手直到最终胜利

## 核心机制

### 1. 花色效果

牌面点数不超过 21 时，按花色触发效果：

| 花色 | 效果 |
|------|------|
| 红桃 (Hearts) | 恢复生命 |
| 方片 (Diamonds) | 造成伤害 |
| 黑桃 (Spades) | 获得防御（吸收方片伤害，回合结束时清零） |
| 草花 (Clubs) | 获得筹码 |

### 2. 点数对照表

| 点数 | 21点计算 | 结算效果值 | 筹码值 |
|------|----------|-----------|--------|
| A | 1 或 11 | 15 | 75 |
| 2-10 | 2-10 | 2-10 | 10-50 |
| J-K | 10 | 11-13 | 55-65 |

### 3. 结算规则

- 双方各可见自己的手牌和对方的第一张牌，后续牌不可见
- 卡牌逐张结算，玩家可自行排序
- 结算顺序：按排序后位置交替结算，由结算先手方开始。结算先手方由双方最终手牌点数决定：点数高者先结算；点数相同则比最大单卡（A=11, 2-10=面值, J/Q/K=10）；仍相同则抛硬币，后手方获 20 筹码补偿
- 发牌和要牌的先后顺序由轮换决定（每回合轮换，初始抛硬币），后手方获得 50 筹码补偿
- 发牌先手与结算先手是两个独立概念：发牌先手影响发牌/要牌顺序，结算先手影响结算交替顺序
- 每张卡按花色触发效果（红桃回复、方片伤害、黑桃防御、草花筹码），防御在回合结束时清空
- 超过 21 点：无视效果，按点数对自己造成伤害

### 4. 特殊牌型（满足手牌点数 ≤ 21）

| 牌型 | 条件 | 效果倍率 |
|------|------|----------|
| 对子 | 两张牌点数一致 | 这两张 ×2 |
| 同花 | 所有牌花色一致 | 所有牌 ×手牌数 |
| 三条 | 三张牌点数一致 | 这三张 ×5 |
| 三七 | 3 张 7 | 所有牌 ×7 |
| 21 | 手牌点数 = 21 | 所有牌 ×2 |
| 杰克 | 点数 = 21 且为 A + J | 所有牌 ×4 |
| 黑杰克 | 点数 = 21 且为黑桃A + 黑桃J | 直接获胜 |

- 同时满足多个牌型时只触发一个，由玩家选择

### 5. 特殊玩法

| 玩法 | 描述 |
|------|------|
| 双倍下注 | 再抽 1 张牌并停止抽牌，手中所有卡牌基础数值翻倍（印记、卡质、牌型倍率不受影响，爆牌自伤翻倍） |
| 分牌 | 两张起始牌相同时触发，分为两手牌分别玩。分牌后不再触发"杰克"或"黑杰克"，其他牌型独立结算 |
| 保险 | 对方可见手牌为 A 时可触发：消耗 30 筹码或 6 生命购买，同时对手明牌。若对方为杰克/黑杰克则返还消耗且无效化牌型效果；否则消耗不返还 |
| 7边池 | 发牌前向边池投入 10/20/50 筹码。发牌后可见 3 张牌中有 1/2/3 张 7 则获得下注额×2/×5/×15 的赔付；无 7 则失去下注 |
| 赌场战争 | 发牌前向边池投入 10/20/50 筹码。双方 4 张牌中最大点数的牌在你手中则获得下注额×2；平局或对手更大则失去下注 |

### 6. 卡牌印记

| 印记 | 效果 |
|------|------|
| 短剑 | 额外造成 2 伤害 |
| 护盾 | 额外获得 2 防御 |
| 心 | 额外回复 2 生命 |
| 钱币 | 额外获得 10 筹码 |
| 重锤 | 摧毁对方同一结算位的卡牌 |
| 跑鞋 | 结算时优先结算此卡牌 |
| 乌龟 | 结算时最后结算此卡牌 |

### 7. 卡牌卡质

| 卡质 | 效果 | 备注 |
|------|------|------|
| 铜质 | 额外获得 10/15/20 筹码 | III/II/I 级 |
| 银质 | 额外获得 20/28/36 筹码 | III/II/I 级 |
| 金质 | 额外获得 30/40/50 筹码 | III/II/I 级 |
| 钻质 | 额外获得 50/66/82 筹码 | III/II/I 级 |
| 红水晶质 | 仅方片，15%/10%/5% 概率被摧毁，伤害 +3/4/5 | III/II/I 级 |
| 蓝宝石质 | 仅红桃，15%/10%/5% 概率被摧毁，回复 +3/4/5 | III/II/I 级 |
| 祖母绿质 | 仅草花，15%/10%/5% 概率被摧毁，筹码 +15/20/25 | III/II/I 级 |
| 黑曜石质 | 仅黑桃，15%/10%/5% 概率被摧毁，防御 +3/4/5 | III/II/I 级 |

- 卡质等级基于国际宝石分级标准：III 类（有内含物）→ II 类（较少）→ I 类（最纯）
- 可通过商店提纯升级等级

### 8. 商店系统

每回合结束后进入商店一次。

| 服务 | 价格 |
|------|------|
| 恢复生命 | 1 生命 = 5 筹码 |
| 印记 | 重锤 300，跑鞋/乌龟 150，其他 100 |
| 赋质 | 铜 40 / 银 80 / 金 120 / 钻 200 / 宝石质统一 120 |
| 买卡 | 基础价（2→10, 3→15, ... J→55, Q→60, K→65, A→75，黑桃 JQKA +10），带增强的随机卡以基础价 + 增强半价出售 |
| 卖卡 | 总投资的 50%（总投资 = 印记价 + 赋质价 + 累计提纯费），无增强卡不可卖 |
| 提纯 | III→II: 100，II→I: 200 |
| 刷新 | 20 筹码（仅刷新随机商品，不可刷新固定服务） |

- 固定服务：恢复生命、赋质、卖卡
- 随机商品：2 个印记 + 2 张带印记/卡质的卡牌

## 回合流程

```
下7边池 → 下战争边池 → 发牌(各两张) → 结算7边池 → 结算战争边池
→ 选择是否购买保险 → 检查保险生效 → 选择分牌
→ 双倍下注/要牌/停牌(分牌则分别进行) → 排序卡牌 → 结算
→ 判断存活
```

### 结算卡牌顺序

```
弹出数值 → 印记效果 → 卡质效果 → 牌型效果 → 计算并执行 → 判断是否被摧毁
```

## 胜负条件

- **对方死亡**: 你获胜，结算筹码，进入商店
- **你死亡**: 游戏结束
- **最终胜利**: 击败一定数量的对手
